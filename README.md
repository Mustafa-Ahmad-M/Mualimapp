# تطبيق «مُعلّمي» — النسخة 2.1

مساعدك الذكي لتحضير وتنظيم حلقات تعليم الأطفال (٦–١٣ سنة). يعمل على الآيفون كتطبيق حقيقي، وبياناتك تتزامن تلقائيًا مع السحابة فلا تضيع أبدًا.

## ✅ كل شيء مُعدّ مسبقًا
- مفتاح **Gemini المجاني** مثبّت داخل التطبيق — الأدوات الذكية تشتغل فورًا (الموديل: gemini-2.5-flash).
- بيانات **Supabase** (الرابط والمفتاح) مثبّتة — المزامنة تلقائية.
- تقدر تغيّر أي مفتاح من **حسابي ← إعدادات الذكاء الاصطناعي**، والتغيير يثبت بعدها.

## ☁️ خطوة واحدة فقط (مرة واحدة للأبد) — تفعيل المزامنة
المزامنة تحتاج جدولًا في قاعدة بياناتك يُنشأ مرة واحدة:
1. افتح محرّر SQL في مشروعك: داخل التطبيق ستجد زر **«افتح محرّر SQL في مشروعك»** في الإعدادات (أو الرابط المباشر في رسالة المحادثة).
2. الصق الكود التالي واضغط **Run**:
```sql
create table if not exists app_backups (
  id text primary key, data jsonb, updated_at timestamptz default now()
);
alter table app_backups enable row level security;
create policy "personal access" on app_backups for all using (true) with check (true);
```
3. خلاص — من دلوقتي بياناتك تترفع تلقائيًا وتُسترجع على أي جهاز أو رابط جديد.

## 🔄 كيف تحمي المزامنة بياناتك
- عند كل تعديل تُحفظ نسخة على جهازك **وتُرفع للسحابة** تلقائيًا.
- عند فتح التطبيق (حتى على رابط Netlify جديد) **يسحب آخر نسخة من السحابة**، فلا تضيع بياناتك حتى لو تغيّر الرابط.

## ✨ جديد هذه النسخة
- **خريطة الرحلة من قلب التحضير**: الذكاء الاصطناعي يحلّل محتوى الدرس الفعلي، يقسّم فقراته إلى مراحل، يضع الاستراحات في مكانها وفق أحدث طرق إدارة الحلقات للسن، **ويكتب نصيحة لكل مرحلة + نصائح عامة**. وكلها قابلة للتعديل.
- **بيانات ثابتة ومزامنة تلقائية** (Gemini + Supabase) دون إدخال يدوي.

## 📲 التثبيت على الآيفون
1. ادخل https://app.netlify.com/drop واسحب **مجلّد** التطبيق كاملًا.
2. افتح الرابط الناتج في **Safari** ← زر المشاركة ← **إضافة إلى الشاشة الرئيسية**.
3. (نصيحة) حافظ على نفس الرابط؛ وحتى لو تغيّر، بياناتك ترجع من السحابة تلقائيًا.
